# 실습에서 배운 내용들

## NULL인 경우 분기

- IS NULL 이용
- NVL() 이용

## 날짜, 시간 정보 형식 정하기

- TO_CHAR() 이용
    
    ```sql
    TO_CHAR(T.TRANSACTION_DATE, 'YYYY-MM-DD HH:MI:SS') 거래일
    ```
    

## 따옴표, 쌍따옴표 사용

- 대부분: 따옴표(’’) 사용
- 한글, 특수 문자, 공백 등이 들어가는 AS: 쌍따옴표(””) 사용

```sql
WHERE D.DEPARTMENT_NAME IN ('Department 3', 'Department 4', 'Department 5', 'Department 6')
```

## 소수점 버리기

- TRUNC() 사용
